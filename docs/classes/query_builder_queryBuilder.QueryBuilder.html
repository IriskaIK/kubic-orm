Class QueryBuilder<T>

A class responsible for building SQL queries for a specific model. Extends the base functionality provided by QueryBuilderBase. This class allows you to build complex SQL queries with methods for selection, joins, limits, offsets, and conditions (WHERE, AND, OR).

Type Parameters

  • T extends Model

    The model type that the query is built for, extending the base Model.
